## Description

[Multicollinearity hinders the interpretability](https://www.blasbenito.com/post/multicollinearity-model-interpretability/) of linear and machine learning models.

The R package [**`collinear`**](https://blasbenito.github.io/collinear/), [available on CRAN](https://CRAN.R-project.org/package=collinear), combines four methods for automated multicollinearity management in modelling data frames with numeric and categorical variables.

To know more, please visit [blasbenito.github.io/collinear](https://blasbenito.github.io/collinear/). 10.5281/zenodo.10039489*
